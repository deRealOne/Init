What Is Html ?

Html:- Is Hyper Text Mark-up Language.It is also a standard mark-up language that is use to create web pages. It is not a programming language.

Html is divided into two parts which is the HEAD the BODY.Let's talk about the Head

HEAD: It is the head of the web browser. It has some functions like linking your external files,showcasing your favicon and also displaying the title of your web brouser

BODY: This is the body of the web browser,it is where all the content in the body tag of your code will be visible
